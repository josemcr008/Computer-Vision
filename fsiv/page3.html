Categorization

Why is difficult?

  • Intra-class variability: We can have a lot of differents chairs.
  • Inter-class: Different classes that shares some caracteristics (a hores and a zebra)
  • Camera viewpoint
  • Oclussion: some parts of the objects are not visible
  • Clutter: many objects in the foreground
  • Deformation: objects changing their position

Convolutional Neural Networks

Deep. feed-forward network with convolutional layers. Filsters are learn automatically from images/videos.

Layers : Convolution

2D Convolutional layer

  • Filter size: eg 3x3
  • Numbers of filters
  • Stide: how many pixels you move to one side
  • Padding: valid, same
  • Activation: ReLU. Once we have the convolution done, the output is called feature map or activation map as well. Activation functions:
    • ReLU: if we have a negative value, we change to 0, the postive stays equal
    • sigmoid: the min value is 0 and max is 1

Dense

A.K.A Fully connected layer. Parameter:

  • Numbers of units
  • Activation

The output

Dense layer with softmax activation. As many units as categories.

Sesion 8

Sequential model

Combine the pieces. Tips

  • Several Conv2D+MaxPool2D bocks, incremental number of filters
  • Some FC layers decreasing theis size
  • Final FC layers with softmax
  • Labels: one-hot vector encoding
  • Relation between output layer and this encoding One-hot vector. A one-hot vector is an vector wich only one position is 1 and the others are 0.

The output we recieve is not perfect, for example, we obtain (0,1,0,0,0,1,0). We have to optimize that.


Model has to be compiled. To be set:

  • loss: image -> output -> target. We want to minimize the loss function as much as possible. Loss is how good is out output comparing to the target
  • Optimizer
  • Metrics: we'll use Accuracy
